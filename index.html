В начале работы тебе необходимо создать структуру приложения, делается это с помощью тегов. Тег — это метка у контента. Метки объясняют, что это за контент и о чем он. Но прежде чем начать ознакомся с этими самыми тегами.

Класс(class) тега — это атрибут который задаёт имя любому элементу на странице. Подходит любому тегу, в дальнейшем будет необходим.

<тег class="имя">Содержимое</тег>

Div — это элемент контейнера, используемый для группировки и упорядочивания тегов на веб-странице.

<div class="box">Содержимое</div>

Кнопка(Button) - элемент который позволяет пользователям взаимодействовать с веб-страницей, выполняя определенные действия.

<button class="btn">Кнопка</button>

Тип(type) тега — это атрибут который задаёт тип любому элементу на странице.

<тег type="тип">Содержимое</тег>

Инпут(input) - элемент, который предназначен для создания текстовых полей, различных кнопок, переключателей и флажков.

<input type="text"></input>

Инпут(input) - элемент который предназначен для создания текстовых полей, различных кнопок, переключателей и флажков.

<input type="checkbox"></input>

Заголовк(h1) - элемент предназначенный для объявления заголовка.

<h1>Текст</h1>
